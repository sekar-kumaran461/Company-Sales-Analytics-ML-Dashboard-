# ================================
# TROUBLESHOOTING COMMON ISSUES
# ================================

# Issue: TensorFlow installation fails
# Solution: pip install tensorflow-cpu (for CPU-only version)

# Issue: XGBoost compilation errors
# Solution: conda install xgboost (use conda instead of pip)

# Issue: Plotly not displaying in Jupyter
# Solution: jupyter labextension install jupyterlab-plotly

# Issue: Streamlit not starting
# Solution: Check firewall settings, use --server.port 8502 for alternative port
